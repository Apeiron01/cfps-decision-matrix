# Inclusion criteria
- CFPS / CFE system with a clearly described reactor format.
- Reports at least one quantitative yield or expression metric and a reaction time.
- Provides enough system context to classify format_class and lysate_type.
- Open literature source (journal, preprint, thesis).

# Exclusion criteria
- Purely theoretical work with no reported data.
- Patents without accessible experimental details.
- Missing format definition or missing yield/time metrics.
- Duplicate reports of the same dataset (keep the most complete version).

# Minimum extraction fields
Required for inclusion:
- format_class and/or format_detail
- yield_value + yield_unit
- time_h
- lysate_type
- paper_id or doi
